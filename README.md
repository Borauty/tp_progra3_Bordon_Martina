Aplicación web de e-commerce desarrollada como Trabajo Práctico para la materia Programación III. Permite a los usuarios navegar productos de tecnología gamer (mouses y teclados), agregarlos a un carrito de compras y finalizar la compra.

Integrantes

- Lautaro Bordon
- Matias Martina

Funcionalidades

- Pantalla de bienvenida con ingreso de nombre de usuario
- Catálogo de productos con renderizado dinámico
- Filtrado por categorías (Mouses y Teclados, Audio)
- Ordenamiento alfabético (A-Z / Z-A) y por precio
- Paginación de productos
- Carrito de compras persistente (localStorage)
  - Sumar y restar cantidad de cada producto
  - Eliminar productos individualmente
  - Vaciar carrito completo
  - Cálculo automático del total
- Finalización de compra con ticket de confirmación (En proceso)
